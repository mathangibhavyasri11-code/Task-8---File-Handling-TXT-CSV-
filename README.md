# Task 8 - File Handling (TXT & CSV)

## Objective
This project demonstrates file handling in Python including:
- Creating and writing text files
- Reading files
- Appending data
- Handling exceptions
- Working with CSV files using csv module

## Tools Used
- VS Code

## Files Included
- file_handling_txt.py
- file_handling_csv.py

## Concepts Covered
- open()
- read(), write()
- append mode
- try-except
- csv module
- with statement
